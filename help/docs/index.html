Introduction and Installation

What is FireNotes?

FireNotes is a notepad application that you can open anytime, anywhere. No matter where you are on the web, you can always consult and edit your notepad. Plus, you can use useful features like dark mode or the burn button. Whoever you are, FireNotes will be a great help to you!

How and where can I install FireNotes?

FireNotes is currently available for Chrome, Microsoft Edge, Opera (And Opera GX) and Mozilla Firefox. If you don't have one of these browsers, you can always use the web version. Visit the download page below to download FireNotes to your browser.

Is FireNotes available for other browsers?

FireNotes can also be installed on other browsers, such as Vivaldi, Brave, and other Chromium-based browsers. You just have to go to the Chrome Web Store and download it conventionally.

Is FireNotes compatible with Safari?

No, FireNotes is not compatible with Safari, but you can install another macOS compatible browser to use FireNotes. You can also use the web version with Safari.

How do I open FireNotes?

Once the extension is installed, click the extension button on the taskbar (üß©), and the menu with all your extensions will open, including FireNotes. Click on FireNotes, and the pop-up with the program will open, and the text you were working on last time will automatically load. If you prefer, you can open it as an external window with the window icon, located in the toolbar of FireNotes. To close the pop-up, simply click outside of it or press the "Esc" key.

Tip: If you want the button to launch FireNotes to be on the taskbar, go to the extensions menu and click the pin icon (The eye icon if you use edge). If you use Firefox: Go to the extensions menu and right-click on FireNotes, then activate "Pin to taskbar."

Opera and Opera GX users: The FireNotes icon is located in the sidebar. To open it, click, and it will open in the side panel.

Open FireNotes in the side panel/sidebar

You can open FireNotes in the sidebar and keep it open, so you can edit and read while visiting your web pages. The process to open FireNotes in this mode varies depending on the browser you use:

Chrome: Press the side panel icon, located in the upper right. Once pressed, the side panel will open. To display FireNotes in the side panel, open the side panel dropdown and select FireNotes.

Edge: Open the extension's menu, find the FireNotes extension, and tap the icon to open it in the sidebar, located next to the extension title. You can also right-click on the extension and select the option to open it in the sidebar.

Opera: In Opera by default it opens in the side panel when you open it normally. You don't have to do anything else.

Firefox: The quickest way to open the sidebar in Firefox is by using the key combination "Ctrl + Alt + N" ("Command + Option + N" if you use Mac). You can use this same combination to close the extension at any time.

How do I uninstall FireNotes?

Go to the extensions menu and click the option menu, then select "Remove extension."

Tip: If you want to delete all data and reset the program to the default state, you don't need to uninstall the extension. You can use the "Reset" button in the FireNotes settings (‚öôÔ∏è). If you want to temporarily stop using the extension without losing data, you can deactivate the extension.

Usage and Features

How to start writing

To start writing in your blank notepad, just click on the empty area. Once you've done that, you can write whatever you want in the notepad.

How to save my text

FireNotes automatically saves everything you write, without pressing save buttons. Whenever you start the program, it will automatically load the last text you were working on. If you want to start over, you can delete everything manually using the "Reset" button in the settings (‚öôÔ∏è) or using the burn text button (üî•). We'll talk more about this last option later.

Increase or decrease text size

You can always increase or decrease the text size using the magnifying glass buttons (üîé). Use + to increase and - to decrease. When you start the program, the text size will automatically adjust to what you had last time. If you want to return to the default size, use the "Reset" button in the settings (‚öôÔ∏è), although keep in mind that other data will be lost.

Burn text

If you click the "üî•" burn text button, all your text will be deleted, but don't worry, it's still safe because the text you were editing has been copied to the clipboard. This means you can paste the text wherever you want in the browser or the system using Control + V (Command + V on macOS) or right-click and select "Paste." If you pressed it by accident, don't worry, you can paste the text back into the notepad and continue working.

Choose a font

In the header, you'll see a dropdown next to the icons. There you can choose the font you want to use in your documents. For now, you can choose from these fonts: Arial, Georgia, Monospace, and Italic. The program will always remember the font you used last time.

ChatGPT

Clicking on the ChatGPT icon will open an external window with the ChatGPT website. This might be useful if you want to ask questions as you write. You can also use the FireNotes notebook to paste AI-generated responses for more convenient editing and viewing.

Settings and Customization

How to open the Settings menu

To open the Settings menu, simply click the gear button (‚öôÔ∏è), which is located in the upper right corner of the editor. All the features we'll go over in this section are inside this menu.

Reset

This function will delete all the application's data and return it to the default state. When you click the button, it will ask if you really want to reset everything. If you accept, the text you were working on will be deleted without the possibility of recovery, so it's advisable to move that text to another place before resetting. It will also deactivate dark mode, reset text size and font to the default state, and choose the default language of the browser.

Dark Mode

If you plan to work for a long time, are in a dark environment, or just want to give your notepad a special touch, you can activate dark mode. When activated, the entire program will have dark colors.

UI Sounds

The FireNotes extension plays small sound effects when performing certain actions, such as burning text, activating or deactivating options, or to alert you of warnings and errors. If you want to disable them, you can uncheck the "Sounds" box in the settings menu.

Interface Language

You can change the interface language at any time. When you first load the program, it will choose the default language of the browser. If it's not available, English will be chosen.
